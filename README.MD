## Install

```shell
yarn add plugin-comic-pro
```

## use

```js
import { Comic, Suppliers } from "plugin-comic-pro";

const cm = new Comic();
const nettruyen = cm.build(Suppliers.NetTuyen, "https://nettruyenco.vn");
```
